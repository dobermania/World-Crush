 World Crush is my first C project: a match-3 game developed without any graphics library, with all visuals rendered directly in the Windows console using ANSI codes. This project was primarily a way for me to learn the fundamentals of the C language, programming logic, and the overall organization of a more complex program. As a result, some game development conventions and structural best practices are not always fully followed, but the project reflects a genuine process of learning and technical growth.
